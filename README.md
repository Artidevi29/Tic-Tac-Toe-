# 🏆 Ultimate Tic‑Tac‑Toe

An advanced, feature‑packed Tic‑Tac‑Toe game built with Python and Streamlit. Play against an **unbeatable AI** (Minimax algorithm) or challenge a friend in two‑player mode. Enjoy match scoring, persistent match history, win animations, and a polished UI.

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🤖 **Unbeatable AI** | Uses the Minimax algorithm – you can only tie or lose |
| 👥 **Two‑Player Mode** | Play locally with custom player names |
| 🏅 **Match Scoring** | Winner starts the next round, tie alternates starter |
| 🏁 **Continue / Quit** | After each round, choose to continue the match or quit |
| 📜 **Match History** | Saves every match (date, scores, winner) to a JSON file |
| 🎉 **Animations** | Balloons for player win, snow for bot win |
| 🎨 **Modern UI** | Emoji icons (❌, ⭕), styled buttons, responsive layout |
| 🌐 **Web App** | Deployed on Streamlit Cloud – play from any browser |

---

## 🕹️ How to Play

1. Choose game mode: **Two Player** or **vs Bot**
2. Enter player names (optional)
3. Click any empty cell to place your mark (❌ or ⭕)
4. The game automatically switches turns
5. After each round, decide to **continue** or **quit** the match
6. Final match winner is shown, and history is saved
